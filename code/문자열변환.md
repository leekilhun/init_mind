## C 문자열 변환  

1. C strtof() 실수 문자열을 float형 실수로 변환
```c
#include "stdlib.h"

char *str;
char *endptr;
double value;

str = "20202.03.12 abc";
value = strtof(str, &endptr);
```
- value에는 2020.03 이 들어가고  
- endptr에는 숫자로 변환하지 못하는 문자열의 시작 위치를 구해 줍니다.  
- 형태 : float strof(const char* str, char ** endptr)
- 인수 :
     . str 실수로 변환할 문자열  
     . endptr 숫자로 변경하지 못하는 문자열의 시작 위치
- 반환 : float형 실수
